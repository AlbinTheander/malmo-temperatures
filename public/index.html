Malmö Temperature Server

This server provides temperature data from Malmö from 1990-01-01 to 2011-12-31. The temperature is measured every third hour.

To access data from specific dates, you query the database like this: 

		http://malmo-temp.herokuapp.com/temp?from=<fromdate>&to=<todate>

Dates should be on the form yyyy-mm-dd

The result is a json array containing temperature entries on the form: 

		{
			timestamp: 1288663200000,
			temperature: 6.1
		}
